General importer for excel files to Postgres

# Requires:

- Python 3.10
- pipenv

## Add pipenv

(Pipenv: Python Development Workflow for Humans)[https://pypi.org/project/pipenv/]

```ps
pip install pipenv
```

## run pipenv

```ps
pipenv run python main.py
```

# .env

LOG_LEVEL=debug
DB_HOST=host name
DB_PORT=port
DB_USER=username
DB_PASSWORD=password
DB_NAME=databae name
